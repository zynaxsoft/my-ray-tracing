# Ray Tracing

Ray tracing in python just for fun!

# Requirement

* PyGame

# Usage

`python3 demo.py -[args]`

## Arguments
  * `-n NUMBER_OF_RAYS, --number-of-rays NUMBER_OF_RAYS`
  * `-o NUMBER_OF_OBJECTS, --number-of-objects NUMBER_OF_OBJECTS`
  * `--width WIDTH`
  * `--height HEIGHT`
